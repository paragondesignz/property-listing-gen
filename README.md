PropAI is a high-performance Micro SaaS designed for real estate agents and property managers. It takes raw property data (starting with the address) and generates compelling, high-converting marketing descriptions using AI.

## Features
- **Address-to-Listing:** Generate a full description just from an address.
- **Tone Selection:** Choose between Professional, Luxury, or Entry-level tones.
- **Key Highlight Extraction:** Automatically emphasizes features like location, views, and modern amenities.

## Tech Stack
- **Frontend:** Next.js 15+ (App Router)
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Hosting:** Optimized for Vercel

## Getting Started
1. Clone the repo.
2. Run `npm install`.
3. Set your `OPENAI_API_KEY` (or Gemini) in `.env.local`.
4. Run `npm run dev`.
